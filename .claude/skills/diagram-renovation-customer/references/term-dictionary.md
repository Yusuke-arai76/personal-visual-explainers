# お客様語 変換辞書（補助金・リフォーム）

## 変換の原則

1. **行政語・業界語は全滅させる**: 最後まで「申請」「交付」「対象工事」などを使わない
2. **お客様の言葉で言い換える**: 「手続き」「もらえるお金」「対象になる工事」
3. **初出に必ず解説ボックスを置く**: 制度名だけは一度だけ正式名称を書いて「この説明書では〇〇と呼びます」と断る

---

## よく使う変換表

### 補助金・助成金の仕組み

| 行政語・業界語 | お客様語 | 使える例文 |
|--------------|---------|-----------|
| 補助金 / 助成金 | 工事費の一部を行政が出してくれるお金 | 「工事費の一部を国が肩代わりしてくれます」 |
| 交付 | もらう / 受け取る | 「お金が受け取れる」 |
| 申請 | 手続き / 申し込み | 「事前に申し込みが必要です」 |
| 申請期間 | 受付期間 | 「〇月〇日まで受け付けています」 |
| 交付決定 | 「使っていいよ」のお知らせ | 「行政から「OK」が届いてから工事を始めます」 |
| 実績報告 | 工事完了の報告 | 「工事が終わったことを書類で伝えます」 |
| 上限額 | もらえる金額の最大 | 「最大〇〇万円まで受け取れます」 |
| 補助率 | 工事費のうち何%か | 「工事費の〇〇%が戻ってきます」 |
| 対象経費 | 補助の対象になる工事費 | 「この工事費が対象になります」 |
| 事前申請 | 工事前に申し込む制度 | 「工事を始める前に手続きが必要です」 |
| 事後申請 | 工事後に申し込む制度 | 「工事が終わってから申し込む制度です」 |
| 予算上限 / 予算額 | 国が用意した全体の予算 | 「予算がなくなり次第締め切りになります」 |
| 性能基準 | 補助を受けるために必要な窓の性能 | 「この基準を満たす窓を選ぶ必要があります」 |

### 工事・商品

| 業界語 | お客様語 | 使える例文 |
|--------|---------|-----------|
| 内窓（二重窓） | 今の窓の内側にもう一枚窓を付ける工事 | 「内側に窓を足すだけなので、工事が短時間で終わります」 |
| 外し替え / 既存枠撤去 | 今の窓を丸ごと取り換える工事 | 「窓ごとまるっと新しくします」 |
| 建具 | 窓・ドアなどの開口部の部品全体 | 「窓やドアのことです」 |
| 断熱性能 / U値 | 外の寒さ・暑さを遮る力 | 「冬の冷気・夏の熱が入りにくくなります」 |

## 解説ボックス HTML 例

制度名など、どうしても正式名称が必要な語は以下の形で初出に置く。

```html
<section class="rounded-2xl border border-ads-border bg-ads-surface p-5 mb-8">
  <div class="flex items-start gap-3">
    <div class="w-10 h-10 rounded-xl bg-blue-50 flex items-center justify-center flex-shrink-0">
      <i data-lucide="book-open" class="w-5 h-5 text-ads-accent"></i>
    </div>
    <div>
      <h3 class="text-lg font-bold text-ads-text">「先進的窓リノベ事業」って何？</h3>
      <p class="mt-2 text-ads-muted leading-relaxed">
        国が省エネ住宅を増やすために作った制度で、窓を断熱性能の高いものに交換する工事に対し、
        <span class="font-semibold text-ads-text">工事費の一部を国が肩代わりしてくれます</span>。
        この説明書では「窓の補助金」と呼びます。
      </p>
    </div>
  </div>
</section>
```

### 注意

- 金額・期限・受付状況は変わりやすい。必ず「〇年〇月時点」「公式サイトで最新情報をご確認ください」と添える
- 「必ずもらえます」「誰でも対象です」という断定は絶対に書かない
