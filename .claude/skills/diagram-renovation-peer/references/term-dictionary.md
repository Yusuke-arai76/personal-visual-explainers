# 業界語・行政語 → やさしい言葉（リフォーム／窓・ドア）

## 変換の原則

1. **お客様のメリットで言い換える**: 「制度上〜」より「いくら安くなる可能性があるか」「何が楽になるか」
2. **誰が・いつ・何をするか**で分解する（助成金・補助金は特に）
3. **初出に用語ボックス**を置き、この図解内での呼び方（略称）を1つに固定する

---

## よく使う変換表（starter）

窓・ドア・リフォーム全般に使いやすい語から。トピックに応じて表を増やす。

### 商品・工事

| 業界語 | やさしい言葉 | お客様向けの例 |
|--------|-------------|----------------|
| 建具 | 窓やドアなど、建物の「開口部」の部品 | 「サッシとガラス、ドア枠などをまとめて建具と言います」 |
| サッシ | 窓の枠と金具のセット | 「窓の骨組みの部分です」 |
| 内窓（二重窓） | 今の窓の**内側**にもう一枚窓を付ける方法 | 「外し替えより工事が軽いことが多いです」 |
| 外し替え | 今の窓を外して、新しい窓に取り換える | 「工事期間や費用は案件次第で変わります」 |
| 断熱性能 / U値 | 窓から熱が逃げにくい度合い（数字は小さいほど断熱が良い、など説明） | 「冬の冷気・夏の熱を抑えやすくなります」 |
| リフォーム | 住みながら、設備や内装・外装を改めること | 「新築ではなく、今の家をよくすることです」 |

### 営業・現場

| 業界語 | やさしい言葉 | お客様向けの例 |
|--------|-------------|----------------|
| 現調 | お宅に行って寸法・状況を確認すること | 「正しいお見積りのために現地を見ます」 |
| 見積 | 内容と金額の提案書 | 「この内容なら、だいたいこの金額、というお話です」 |
| 発注 | お客様が正式に依頼すること | 「ご契約・ご注文いただいた後のスケジュールは…」 |
| 工期 | 工事にかかる日数の目安 | 「天候や材料で前後することがあります」 |

### 行政・制度（汎用）

制度ごとに必ず一次情報で確認すること。**図解に載せる条件は出典とセット**。

| 行政語 | やさしい言葉 | たとえ |
|--------|-------------|--------|
| 助成金 / 補助金 | 条件を満たすと**行政が一部負担してくれる**仕組み（制度ごとに違う） | 割引クーポンだが、使える店・条件が決まっている |
| 対象者 | 申請できる人・事業者の条件 | 年齢・所得・地域などの「入場条件」 |
| 対象工事 | お金が出る工事の範囲 | メニュー表の「対象商品」 |
| 申請期間 | 受付の開始・終了 | キャンペーンの締切 |
| 上限額 | もらえる（戻る）金額の天井 | 割引の上限 |
| 事前申請 | 工事の前に承認を取る必要 | 旅行のビザ申請のように「先にOKが要る」 |
| 事後申請 | 工事や支払いの後に手続き | レシートを送ってキャッシュバック |
| 性能基準 | 窓の性能など、満たすべき数値・規格 | 試験の合格ライン |

## 用語解説ボックス（HTML例）

`base.html` 内のコンテンツでは、次のようなカードを使う。色は ADS の `ads-accent` / 薄い背景で統一してよい。

```html
<section class="rounded-2xl border border-ads-border bg-ads-surface p-5 mb-8">
  <div class="flex items-start gap-3">
    <div class="w-10 h-10 rounded-xl bg-blue-50 flex items-center justify-center flex-shrink-0">
      <i data-lucide="book-open" class="w-5 h-5 text-ads-accent"></i>
    </div>
    <div>
      <h3 class="text-lg font-bold text-ads-text">「U値（ユーち）」とは？</h3>
      <p class="mt-2 text-ads-muted leading-relaxed">
        窓の<span class="font-semibold text-ads-text">断熱の良さを数字で表したもの</span>です。一般にお客様には「小さいほど外の熱・冷気の影響を受けにくい」と説明することが多いですが、
        <strong>制度で数値が決まっている場合は、その制度の定義に合わせて</strong>お伝えください。
      </p>
      <p class="mt-2 text-sm text-ads-dim">この図解では「断熱の成績表」と呼ぶこともあります。</p>
    </div>
  </div>
</section>
```

### 注意（営業コンプライアンス）

- 金額・受付状況・残予算は**変わりやすい**。図解には「〇年〇月時点」「要確認」と明記する
- 確定していない内容を**約束口調**にしない
